# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-05-26

### Added

- Secure PowerShell sync baseline for Snipe-IT and Microsoft Graph.
- Dry-run support and `ShouldProcess` integration.
- Explicit create, update, archive, and delete switches.
- Destructive delete confirmation guardrail.
- Structured JSONL logging with redaction.
- JSON summary report generation.
- Retry handling for transient API failures.
- Microsoft Graph pagination support.
- Snipe-IT pagination support.
- Duplicate and bad-serial detection.
- Example configuration and JSON schema.
- Operational runbook.
- Pester baseline tests.
- GitHub Actions CI workflow.
- Repository `.gitignore` for local secrets and runtime output.
